# 管理员管理

商城管理员进入管理员成员管理程序后，可以对管理员进行管理；
其中管理员分为两类：超级管理员（可以操作一切，但是超级管理
员之间不能互相操作，超级管理员可以添加超级管理员，可以修改部分信息或删除普通管理员，注：添加之后不可修改或删除超级管理员，需慎重），普通管理员（需要赋予对应的角色，根据角色赋予对应的权利，不建议普通管理员操作核心模板）
